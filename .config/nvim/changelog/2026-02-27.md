# 2026-02-27: Migrate to lazy.nvim

Replaced vim-plug and packer.nvim (archived) with lazy.nvim as the single plugin manager.

## Changes

- **lua/plugins.lua**: Rewrote from packer to lazy.nvim bootstrap + specs. All 32 plugins from both managers consolidated here. Moved nvim-tree `on_attach` inside its config block. Moved spectre keymaps into `keys` field for lazy-loading.
- **init.vim**: Removed `plug#begin()`/`plug#end()` block and `packer_user_config` augroup.
- **plugin/packer_compiled.lua**: Deleted (auto-generated packer artifact).

## Lazy-loaded plugins

- goyo.vim (`cmd = "Goyo"`)
- tabular (`cmd = "Tabularize"`)
- markdown-preview.nvim (`ft = "markdown"`)
- vim-which-key (`cmd`)
- nvim-spectre (`keys`)
